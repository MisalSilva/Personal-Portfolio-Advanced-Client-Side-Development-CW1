Plane Management System (Ticket Booking System)

This project is a plane management system developed for the semester 2 coursework of the computer science degree program. The system allows users to:
1. Book plane tickets
2. Cancel plane tickets
3. Find the first seat that is available to buy
4. Display seating plan with available and bought seats marked with O and X
5. Search any bought ticket and view print tickets information and total sales

Progress Outcome Predictor

This progress outcome predictor was developed to predict students' progress in a university at the end of each academic year. Finally, the progress outcome was recorded in a histogram.
1. The program prompts for the number of credits at pass, defer and fail.
2. Then the program displays the appropriate progression outcome for an individual student which is progress, trailing, module retriever or exclude.
3. The user can enter more credit marks or exit the predictor and display a summary of the progess record in a histogram.
